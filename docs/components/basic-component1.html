创建目录

win键+R键，输入 cmd，打开命令终端 先进入任意盘符，比如 F 盘

sh
#盘符可以自定义 回车进入
f:
1
2

再创建文件夹名并进入

sh
#创建目录并进入文件夹
mkdir vitepress && cd vitepress
1
2

这样我的目录路径为 F:\vitepress 我们先关闭cmd，一会介绍快捷进入的方法

安装 ​

安装依赖 ​

在项目目录上方的地址栏，上输入 cmd 回车可以快捷打开

然后我们安装vitepress

sh
pnpm add -D vitepress
1
sh
yarn add -D vitepress
1
sh
npm i -D vitepress
1
sh
bun add -D vitepress
1

sh
